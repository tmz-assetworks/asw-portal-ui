## **Running ASW Rest Frontend**

#### ● **Prerequisites**
1. Before you begin, ensure you have met the following requirements:
2. You have a basic understanding of node and npm.
3. You have Nodejs installed with version 16.14.2 or greater.
4. You have angular cli installed with version 13.3.7. If not installed, you can install
   Angular CLI globally with NPM:
   a. npm install -g @angular/cli
   b. After it&#39;s installed, confirm installation:
   ng --version

#### **● Configuring and running the application***
1. Git clone from the repository.
2. cd &lt;base_code_folder&gt;.
3. To configure backend server URL:
   a. cd src/environments/environment.ts
   b. Set value for “serverUrl:: http://&lt;IPaddress&gt;:&lt;port&gt;,
   c. Run `npm install` to install all dependencies and packages.
   d. Run `npm start` to start the application `http://localhost:4200/`
   e. Run `ng build`
